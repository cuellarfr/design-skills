UX Research

From interview scripts to usability tests to workshop facilitation. Includes a catalog of 20+ methods, SUS scoring with the full 10-question scale, and Opportunity Solution Trees for continuous discovery.

Design Critique & Evaluation

A 7-step critique framework that uses Nielsen's heuristics, 27 UX laws, and 50+ interaction patterns to give feedback that's specific, not subjective. Includes severity scales and scoring rubrics.

10 files · 2,384 lines

Accessibility Audit

Five audit layers — automated, keyboard, screen reader, visual, and flow — against WCAG 2.2 at A, AA, and AAA. Flags issues, rates severity, suggests code fixes, and generates reports.

8 files · 2,175 lines

Journey Mapping

Seven diagram types — journey maps, service blueprints, experience maps, empathy maps, and more. Covers service ecology, experience prototyping, and the SERVQUAL measurement framework.

Design Systems

The full lifecycle — from interface inventory through token architecture, component specs, pattern documentation, and governance. Covers Atomic Design, Kholmatova's pattern approach, and the W3C token format.

UX Strategy

How things should behave, not just how they look. Microinteraction specs, state machines, product postures, motion timing, and a 6-level error prevention hierarchy. Based on Cooper and Saffer.

Installation

All skills

# Clone the repo
git clone https://github.com/cuellarfr/design-skills.git

# Copy all skills to Claude Code
cp -r design-skills/*/ ~/.claude/skills/

Single skill

# Example: install only design critique
cp -r design-skills/design-critique \
  ~/.claude/skills/
How skills are structured
skill-name/
├── SKILL.md              # Main file — works standalone
│
├── references/           # Deep reference material
│   ├── topic-a.md
│   └── topic-b.md
│
├── templates/            # Fillable deliverable templates
│   └── report-template.md
│
└── examples/             # End-to-end walkthroughs
    └── walkthrough.md
  • Opinionated Specific numbers and rubrics, not "consider..." language
  • Actionable Every principle comes with a concrete recommendation
  • Show, don't tell Before/after examples with scoring throughout
  • Depth on demand Start with the main file. Pull in references when you need more
  • Real sources 15 books, 11 frameworks, 6 standards. Nothing made up
